# Security Notes

This repository is a sanitized public showcase version.

The following content is intentionally excluded or replaced with placeholders:

- WeChat Mini Program private project config files.
- Real WeChat `appid`.
- Real Tencent CloudBase environment ID.
- API keys and local environment files.
- Local IDE cache, dependency folders, logs, cookies, and generated build outputs.

Before deploying, copy `.env.example` to your own environment variable setup and configure:

- `HUNYUAN_ENV_ID`
- `HUNYUAN_API_KEY`

Do not commit real keys, private config files, user data, chat logs, or cloud database exports.
